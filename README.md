Simulación Kiosko de Triage (academico)

Descripción

Pequeña aplicación en Streamlit que simula un kiosko de triage para atención hospitalaria (prototipo académico). Permite identificar/registrar pacientes, ingresar o simular signos vitales, calcular triage y generar tickets.

Instalación rápida

1. Crear y activar un entorno virtual (Windows PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2. Ejecutar la app:

```powershell
streamlit run app.py
```
